#AWS #Service #Migration 
### AWS Application Migration Service (MGN)

Fully managed service for lift-and-shift migrations of on-premises servers (physical, virtual, cloud) to AWS EC2 with minimal downtime. Automates continuous block-level replication, testing, and cutover. Supports OS like Windows/Linux; ideal for large-scale migrations.

### Related Services

- [[AWS Migration Hub]]: Centralizes tracking of MGN progress.
- Amazon EC2: Hosts migrated servers and applications.
- [[AWS Application Discovery Service]]: Discovers on-premises servers for planning.
- [[Amazon CloudWatch]]: Monitors replication metrics and performance.
- [[AWS CloudFormation]]: Provisions resources post-migration.

### Related Concepts

- Lift-and-Shift Migration: Relocates apps to cloud without redesign.
- Continuous Replication: Block-level sync for near-zero data loss.
- Cutover: Final switch to AWS environment.
- Application Modernization: Optimizes workloads after migration.
